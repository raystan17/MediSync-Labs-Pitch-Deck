# MediSync Labs — LAUNCH Co. Pitch Deck

Interactive pitch deck for MediSync Labs. Deployed for easy sharing with investors and partners.

**Live:** Deploy to Vercel (see below) for a shareable link.

## How to view

- **Online:** Open the deployed Vercel URL (see below)
- **Local:** Open `index.html` in any browser

**Navigation:** Use ← → arrow keys, or click left/right side of the screen.

## Deploy to Vercel

1. Push this repo to [GitHub](https://github.com/raystan17/MediSync-Labs-Pitch-Deck)
2. Go to [vercel.com](https://vercel.com) and sign in with GitHub
3. Click **Add New** → **Project**
4. Import `raystan17/MediSync-Labs-Pitch-Deck`
5. Click **Deploy** (no build config needed — it's static HTML)

Your deck will be live at `medisync-labs-pitch-deck.vercel.app` (or a custom domain).
